## Lessons (for next plans)

1. **Dockerfile toolchain bumps** need a rebuild-without-cache runbook — stale
   Node layers look like app bugs.
2. **Docs cited by rules** must exist or `check_rule_health.sh` and onboarding
   drift — close reference doc gaps in the same release.
3. **Handler integration shard** needs at least one test file or CI matrix slot
   is hollow.
